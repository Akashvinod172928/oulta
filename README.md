## 🚦 Getting Started

This project is a Flutter application. Follow these steps to get started:

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code
- iOS development tools (for Mac users)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Akashvinod172928/oulta.git
   cd oulta
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```
